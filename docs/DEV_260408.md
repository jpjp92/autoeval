## KO 프롬프트 Fallback 동기화 (별도 트래킹)

`generators/prompts.py`의 KO Fallback(`SYSTEM_PROMPT_KO_V1`, `USER_TEMPLATE_KO_V1`)이 주 경로(P3~P8) 패치를 미반영한 채로 유지 중.

| 패치 | 주 경로 | KO Fallback |
|------|---------|-------------|
| P1 (comparison ≤40%) | ✅ | ✅ |
| P2 (변화방향) | ✅ | ✅ |
| P3 (purpose조건) | ✅ | ❌ 미반영 |
| P4 (추론금지) | ✅ | ❌ 미반영 |
| P5 (완전성) | ✅ | ❌ 미반영 |
| P7 (대칭검증) | ✅ | ❌ 미반영 |
| P8 (카테고리중의성) | ✅ | ❌ 미반영 |

> Fallback은 `qa_generator.py`에서 `system_prompt=None`일 때만 사용. 정상 경로에는 영향 없으나 fallback 품질 저하 가능성 존재.
