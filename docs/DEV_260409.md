## FIX-A: `detect_heading()` — 패턴 매칭 font 하한선

### 문제

running page header (각 페이지 상단 반복 출력 챕터 제목, font 8~9)가 y축 필터(5%)를 통과한 뒤
`^제\s*\d+\s*장` 등 패턴에 매칭되어 heading으로 오분류됨.

```
[p06] font=8.0  "제1장 작성 배경 및 목적"   ← running header, 매 페이지 반복
[p08] font=8.0  "제1장 작성 배경 및 목적"   ← 동일
...
```

### 수정 (`backend/ingestion/parsers.py:199`)

```python
# 변경 전
is_pattern_match = any(re.match(p, clean_text) for p in heading_patterns)

# 변경 후
is_pattern_match = (
    any(re.match(p, clean_text) for p in heading_patterns)
    and font_size >= 10.0   # running header(font 8~9) 오분류 방지
)
```

> `is_font_boost` 조건(`font_size > 12.5`)은 이미 하한이 있으므로 변경 없음.

## FIX-B: `extract_text_by_page()` — `_KOR_CONT` y_gap 가드

### 문제

`_KOR_CONT`/`_KOR_TAIL` 병합 보정 로직이 heading + 본문 첫 문장을 잘못 결합.

```
# 현재 출력 (잘못된 병합)
"1.2.1. 등장배경추론형 AI 데이터는 AI 발전에서 지식 기반 시스템..."

# 기대 출력 (정상 분리)
"1.2.1. 등장배경"
"추론형 AI 데이터는 AI 발전에서 지식 기반 시스템..."
```

**오발동 조건**:
- `prev_text` = `"1.2.1. 등장배경"` → 끝이 한글("경") → `_KOR_TAIL` 매칭
- `cur_text`  = `"추론형 AI 데이터는..."` → 3자 한글+공백 → `_KOR_CONT` 매칭
- → 어절 절단으로 오인, 병합됨

**y_gap 분포 (실측)**:

| 케이스 | y_gap | 판정 |
|--------|-------|------|
| 정상 어절 절단 | 8~10px | 병합 허용 |
| heading + 본문 | 17~35px | **병합 차단** |
| heading 2줄 분할 (예: "제1장" + "총론") | 21px | font_diff=0이므로 통과 |

### 수정 (`backend/ingestion/parsers.py:688`)

```python
# 변경 전
is_kor = (
    _KOR_CONT.match(stripped)
    and _KOR_TAIL.search(prev_text.rstrip())
)

# 변경 후 — y_gap > 15 AND font_diff > 1.5이면 별개 단락으로 간주
is_kor = (
    _KOR_CONT.match(stripped)
    and _KOR_TAIL.search(prev_text.rstrip())
)
if is_kor:
    prev_y1   = prev_block.get("bbox", [0, 0, 0, 0])[3]
    y_gap_val = y_pos - prev_y1
    prev_font = prev_block.get("font_size", 10.0)
    cur_font  = block_dict.get("font_size", 10.0)
    font_diff = abs(prev_font - cur_font)
    if y_gap_val > 15.0 and font_diff > 1.5:
        is_kor = False  # heading+본문 오병합 방지
```

**조건 설계 근거**:
- `y_gap > 15` 단독: heading 2줄 분할(y_gap=21, font_diff=0)을 차단해 회귀 발생
- `font_diff > 1.5` 단독: 같은 font heading+본문을 차단 못함
- **두 조건 AND**: heading(font 14)+본문(font 11)은 gap=30, diff=3 → 차단 / heading 2줄은 diff=0 → 허용
