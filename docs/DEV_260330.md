# 2026-03-30: QA 품질 평가 및 통계 분석 고도화 산출물

이 문서는 QA 품질 평가 엔진의 `완전성(Completeness)` 지표 개선 사항과 `데이터셋 통계(Layer 1-B)`의 산출 방법론 및 지표 정의를 상세히 정리합니다.

---

## 1. 품질 평가: 완전성 (QA Quality: Completeness)

LLM 답변이 질문의 모든 요구사항을 충실히 반영했는지 측정하는 지표입니다.

### 1.1 측정 원리: 질문 분해 (Decomposition)
- **원리**: 복합적인 질문을 원자 단위의 서브 질문(Atomic sub-questions)으로 분해하여 평가의 객관성을 확보함.
- **예시**: "A와 B의 차이점은 무엇이며, 각각의 장점은?"
  - 서브 질문 1: A와 B의 차이점이 설명되었는가?
  - 서브 질문 2: A의 장점이 언급되었는가?
  - 서브 질문 3: B의 장점이 언급되었는가?

### 1.2 채점 기준 및 커버리지 (Coverage)
- **커버리지 계산**: `답변된 서브 질문 수 / 전체 서브 질문 수` 비율로 산출.
- **점수 산출 (Scaling)**: 
  - 커버리지 100%: 10점 (완벽)
  - 커버리지 80% 이상: 8-9점 (사소한 누락)
  - 커버리지 50% 미만: 5점 이하 (핵심 내용 누락)
- **누락 요소 탐지**: 답변에서 다루지 못한 측면을 `missing_aspects`로 추출하여 사용자에게 구체적인 피드백 제공.

## 2. 데이터셋 통계: Layer 1-B 산출 방법론

데이터셋 자체의 건전성과 다양성을 물리적으로 측정하는 지표입니다.

### 2.1 다양성 (Diversity)
*   **정의**: 데이터셋이 얼마나 다양한 주제와 어휘를 포함하고 있는가.
*   **산출 방식**:
    - **인텐트 엔트로피 (60%)**: Shannon Entropy($-\sum p_i \log_2 p_i$)를 통해 인텐트 분포의 균형도를 측정. (균등할수록 고득점)
    - **어휘 다양도 (TTR, 40%)**: 고유 토큰 수 대비 전체 토큰 수 비율($Unique / Total$). (중복 단어가 적을수록 고득점)
*   **목표**: 특정 유형의 질문에만 매몰되지 않은 균형 잡힌 데이터셋 확보.

### 2.2 중복도 (Duplication)
*   **정의**: 서로 다른 질문 쌍 간의 텍스트 유사성 정도.
*   **산출 방식**: `SequenceMatcher` 알고리즘을 사용해 질문 텍스트 간 유사도가 70% 이상인 쌍(Near-duplicate)을 탐지.
*   **감점 로직**: `10 - (중복률 / 2)`. 중복률이 20%를 넘어가면 0점 처리하여 데이터 중복을 엄격히 관리.

### 2.3 편향도 (Skewness)
*   **정의**: 특정 소스 문서(docId)에 질문이 과도하게 집중되어 있는 정도.
*   **산출 방식**: 전체 데이터셋 중 단일 문서가 차지하는 최대 비중(`doc_max_ratio`)을 기준으로 구간 채점.
*   **기준**: 
    - 30% 이하: 10점 (이상적 분포)
    - 50% 이하: 8점
    - 80% 이하: 5점
    - 80% 초과: 2점

### 2.4 충족성 (Sufficiency)
*   **정의**: 데이터 분석 및 학습에 필요한 필수 정보가 물리적으로 갖춰졌는가.
*   **산출 방식**:
    - **필드 채움률**: 질문, 답변, 컨텍스트, 문서ID, 인텐트 등 5개 필수 필드의 기입 여부 전수 조사.
    - **물리적 볼륨**: 질문/답변의 평균 길이를 통해 텍스트가 분석하기에 충분한 정보량을 담고 있는지 보조적으로 판단.

## 3. UI/UX 및 데이터 매핑 원칙

- **용어 통일**: 품질 평가의 **'완전성'**과 통계의 **'충족성'**을 엄격히 구분하여 사용자 혼선 방지.
- **감사 리포트 스타일**: 모든 분석 결과는 [헤더 바 + 액센트 라인 + 컨텐츠] 구조의 리포트 형식으로 시각화.
- **데이터 투명성**: 정량적 점수(Metrics) 뒤에 항상 구체적인 근거 데이터(Stats/Reason)를 배치하여 분석 결과의 신뢰도 확보.
