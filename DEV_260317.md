### 3. QA 생성 전 Content 품질 진단

모든 청크 content 전문을 조회하여 문제 유형 분류.

#### 문제 분류

| 유형 | 청크 | 문제 | 발생 원인 |
|------|------|------|-----------|
| **파싱 아티팩트** | [1][2][4][5] | 문장 중간 줄바꿈 (`필수적이지만, 데이터\n중복`) | PDF "dict" 모드가 시각적 줄 경계를 `\n`으로 출력 |
| **파싱 아티팩트** | [4][5][8] | `Ÿ` 특수문자 불릿 | PDF 불릿 폰트 → 유니코드 변환 실패 |
| **짧은 청크** | [3] (57자), [7] (258자) | QA 생성 불가 수준의 context 부족 | 최소 필터 50자만 있고 섹션 분리가 너무 세밀함 |
| **청크 끝 잔해** | [0][2] | 다음 섹션 표 헤더 조각이 붙음 | 연속된 섹션 경계에서 스플리터 overlap 영향 |
| **무의미 청크** | [11] | 발행처/저작권 페이지 | QA 생성 불필요, 임베딩 낭비 |

#### 설계 판단: Ingestion 수정 vs Generation 필터

```
범용적으로 재현되는 파싱/청킹 문제 → Ingestion 수정
Task/정책 의존적 판단 → Generation 필터
```

| 항목 | 처리 레이어 | 근거 |
|------|-------------|------|
| `Ÿ` → `-` 불릿 정규화 | **Ingestion** | 모든 PDF에서 공통 발생 |
| 문장 중간 줄바꿈 스마트 결합 | **Ingestion** | PDF 줄바꿈 ≠ 문단 경계, 범용 문제 |
| 짧은 청크 병합 (`< 200자`) | **Ingestion** | 임베딩 품질·비용 모두 낭비 |
| 발행처/저작권 페이지 스킵 | **Ingestion** | 어떤 QA task에도 무의미 |
| `heading` 청크 단독 제외 | **Generation** | task에 따라 제목 기반 QA도 유효 |
| L1 균형 샘플링 | **Generation** | 배분 정책은 task별 가변 |

### [Ingestion] `normalize_text` 강화

```python
# 1. Ÿ (불릿 폰트 아티팩트) 정규화
text = re.sub(r'[ŸŸ]', '-', text)  # 유니코드 변형 모두 처리

# 2. 스마트 줄바꿈 결합 (문장 중간 줄바꿈 → 공백)
# 조건: 이전 줄이 .!?。로 끝나지 않고, 다음 줄이 불릿/표/제목이 아닌 경우
def smart_join_lines(text: str) -> str:
    result, buffer = [], ""
    for line in text.split('\n'):
        stripped = line.strip()
        if not buffer:
            buffer = stripped
        elif (buffer
              and not re.search(r'[.!?。]$', buffer)
              and not re.match(r'^[-•|\[#\d]', stripped)):
            buffer += ' ' + stripped
        else:
            result.append(buffer)
            buffer = stripped
    if buffer:
        result.append(buffer)
    return '\n'.join(result)
```

### [Ingestion] 짧은 청크 병합

스플리터 후 후처리 단계에서, 같은 `section_path` 내 인접 청크가 200자 미만이면 다음 청크와 병합 (최대 `chunk_size`까지).

```python
def merge_short_chunks(chunks: list, min_chars: int = 200, max_chars: int = 1200) -> list:
    merged, buf = [], ""
    for chunk in chunks:
        if not buf:
            buf = chunk
        elif len(buf) + len(chunk) < max_chars and len(buf) < min_chars:
            buf += "\n" + chunk
        else:
            merged.append(buf)
            buf = chunk
    if buf:
        merged.append(buf)
    return merged
```

### [Ingestion] 발행처/저작권 청크 필터

```python
COLOPHON_KEYWORDS = ["발행처:", "발행인:", "저작권", "©", "무단전재", "재배포를 금"]

def is_colophon_chunk(text: str) -> bool:
    return sum(1 for kw in COLOPHON_KEYWORDS if kw in text) >= 2
```
